Predicting Mood Based on Spotify Audio Features

  • Built a machine learning model to predict the emotional mood of a song (happy vs. not happy) based on Spotify audio features like energy, danceability, and tempo. Explored data preprocessing, class imbalance handling, and model tuning using Random Forests and Logistic Regression. Achieved solid predictive performance and visualized decision boundaries and feature importance to interpret model behavior. This project demonstrates my ability to combine music analytics with supervised classification and thoughtful evaluation. Tools: Python, Scikit-learn, Pandas, Matplotlib Skills: Classification, Feature Engineering, Model Evaluation, Data Visualization
