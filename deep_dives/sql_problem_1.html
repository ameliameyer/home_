Top Regional Managers SQL Deep Dive

This deep dive walks through a SQL query to identify the top 3 regional managers based on their teams' total sales in the last 6 months.

Video Walkthrough

SQL Problem Statement

Find the top 3 performing regional managers based on the total monthly sales made by their team in the last 6 months.

Approach and SQL Script

The query uses Common Table Expressions (CTEs) to:

  • Calculate total sales by region in the last 6 months
  • Select the top 3 regions by sales
  • Find the regional managers responsible for those top regions

Key SQL elements: CTEs, aggregation, filtering by date, joining employees and sales tables.

Tools and Techniques

  • SQL (T-SQL with CTEs)
  • Data aggregation and filtering
  • Analytical thinking and query optimization
