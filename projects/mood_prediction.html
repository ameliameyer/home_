Mood Classification using Spotify Audio Features

Can machines detect the mood of a song? This project uses Spotify audio features to classify songs as either 'happy' or 'not happy' based on valence score.

Github Repository

🎯 Motivation

Music plays a powerful role in shaping how we feel. I wanted to explore whether machine learning could detect a song's emotional tone based on Spotify's audio features.

📌 Project Overview

  • Cleaned and prepared a dataset of thousands of songs
  • Engineered a binary mood label from Spotify's valence score
  • Trained and compared Logistic Regression, Random Forest, SVM, and XGBoost models
  • Evaluated models using accuracy and F1 score

📊 Results

Random Forest performed the best overall:

Model Accuracy F1 Score
Random Forest0.720.72
XGBoost0.710.71
SVM0.710.71
Logistic Regression0.690.69

Insight: Danceability and energy were key predictors of a song’s mood.
