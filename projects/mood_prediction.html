Mood Classification using Spotify Audio Features

Can machines detect the mood of a song? This project uses Spotify audio features to classify songs as either 'happy' or 'not happy' based on valence score.

Dataset

  • Features: Danceability, Energy, Tempo, Acousticness, Valence
  • Target: Binary label — 1 = Happy (valence > 0.5), 0 = Not Happy

Models Used

  • Logistic Regression
  • Random Forest
  • SVM
  • XGBoost

Results

Random Forest performed best with an accuracy and F1 score of 0.72.

Tools

  • Python (Pandas, Sklearn, Seaborn, Matplotlib)
